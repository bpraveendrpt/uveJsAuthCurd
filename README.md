# vue-3 User curd App

Vue 3 + Pinia - User Registration and Login and add Counries 
# install CLI

npm install -g @vue/cli 

# check version

vue --version

# Create Uve Projects 

vue create <project name>
# Install Node Modules 
npm i install

# Run The Projects

npm run serve
npm run dev 


# install next uve

npm install vuex@next --save
